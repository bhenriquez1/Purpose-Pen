# Purpose Pen — Claude Prompt

> Sections 1–7 of this prompt are defined elsewhere and not yet captured in this file. This document currently holds Section 8, added to extend Purpose Pen from a writing tool into an application management platform.

## 8. Letter of Recommendation Suite

### A. Recommender Packet Generator
- Collect applicant achievements
- Collect volunteer experiences
- Collect shadowing experiences
- Collect work history
- Generate recommender packet

### B. Letter Draft Assistant
- Dental school recommendation letters
- Medical school recommendation letters
- Faculty letters
- Dentist letters
- Employer letters
- Research mentor letters

### C. Recommendation Request Builder
- Generate professional email requests
- Generate follow-up reminders
- Track recommendation status

### D. Letter Analyzer
- Analyze uploaded recommendation letters
- Identify strengths and weaknesses
- Suggest improvements
- Detect generic language

### E. Committee Packet Builder
- Combine recommendation letters
- Resume/CV
- Personal statement
- Achievements summary
- Volunteer summary
- Leadership summary

### F. Recommendation Tracker
- Requested
- In Progress
- Submitted
- Received
- Missing
- Deadline tracking

## Dental School Applicant Workflow

1. Build CV
2. Build Personal Statement
3. Request Letters of Recommendation
4. Track Letter Status
5. Generate Supplemental Essays
6. Export Application Packet

This workflow extends to medical, PA, nursing, graduate, and scholarship applicants as well, positioning Purpose Pen as an application management platform rather than a single-purpose writing tool.
